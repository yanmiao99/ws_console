## 插件作用

> 快速生成 console.log

## 使用方法

> 在 `JavaScript` 文件中使用 `变量.log` 然后按下快捷键即可生成
>
> `console.log('test=========>', test);`

> ⌨️ 快捷键
>
> - mac : cmd + enter
> - windows : alt + enter

```javascript
let test = 123;

// test.log  按下 cmd + enter

console.log('test=========>', test);

// 123.log

console.log('123=========>', '123');
```

## 注意点

> 需要在 `.log` 后面按下快捷键
